# Simple API for todo management

## Installation

1.Install the required packages:
```bash
pip install -r requirements.txt
```

2.Create virtual environment:
```bash
python3 -m venv venv
```

3.Activate virtual environment:
```bash
source venv/bin/activate
```
    
4.Run server:
- In Linux:

```bash
python3 main.py
```
- In Windows:
```bash
python main.py 
```
